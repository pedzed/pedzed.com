Over mij

Ik ben Ped Zed. Graag werk ik met computers. Ik ben een doorzetter die graag resultaat ziet. Vandaar ook de interesse in programmeren. Ik maak niet alleen mooie en werkende web applicaties, maar ik maak ze ook van de allerhoogste kwaliteit.

Zowel in het echte leven als in de IT vind ik security erg belangrijk. Daarom ben ik een grote voorstander van up-to-date blijven, Transport Layer Security, kant-en-klare security libraries en "security on default".

Hobby's

Naast programmeren vind ik het ook erg leuk om te sporten. Om gezond te blijven doe ik wekelijks drie dagen aan judo en Mixed Martial Arts (MMA).

Full-stack web development

Ik ben een full-stack web developer die van alles kan in het gebied van web-based applicaties. Al hoewel mijn expertise in back-end web development ligt met voornaamelijk PHP, kan ik ook goed omweg met de front-end. Ik schrijf niet alleen code in HTML5, CSS3 en JavaScript, maar ik doe ook vaak veel aan design.

PHP

Ik schrijf mijn code altijd zo object-geörienteerd mogelijk. Naast een goede kennis van PHP zelf, heb ik ook ervaring met het Laravel PHP framework.

HTML5, CSS3 & JavaScript

Met ruim vijf jaar ervaring met HTML en CSS kan ik de mooiste web applicaties maken. JavaScript, die ik vaak gebruik voor AJAX, gebruik ik meestal om de user experience te verbeteren. Ik schrijf namelijk zelden code waarbij de website zonder JavaScript onbruikbaar is.

Om designs nog sneller te kunnen schrijven gebruik ik overigens de Foundation framework met SCSS (SASS) en gebruik ik jQuery voor visuele verbeteringen.

Open source

Ik ben een grote voorstander van open source software. Ik doe dus ook wel eens het één en ander voor de open source community. Zo meld ik bijvoorbeeld bugs en help mensen met hun projecten.

Uiteraard werk ik net als ieder ander programmeur met versiebeheer systemen zoals Git. Mijn publieke GitHub pagina is te bereiken op https://github.com/pedzed.

Database design

Ik heb kennis opgedaan met middelmatig-grote relational databases. Zo heb ik bijvoorbeeld een heel rooster systeem gedesigned voor scholen.

Ik heb ervaring met MySQL, SQLite en Microsoft Access. Verder ben ik nog bekend met MySQL Workbench en Oracle database design.

Linux desktop & servers

Ik ben een dagelijkse Linux gebruiker. Wanneer ik aan het programmeren ben, werk ik het liefst met Ubuntu, omdat ik heel comfortabel kan werken met Unity's virtual desktops.

Ik heb ervaring met Ubuntu servers. Zo heb ik enkele Virtual Private Servers beheerd met LAMP stacks.
